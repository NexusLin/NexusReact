
# React + TypeScript + Vite

## 一、项目信息
—————————————————————————————————————————————————————————
### 项目目录以及文件夹含义介绍
public              公共资源
    index.html        首页
    mainfest.json     pwa应用的元数据


src                 项目源码
    assets            资源（图片,字体图标）
    componenets       公共组件
    pages             页面
    utils             工具
    app.js            项目根组件
    index.css         局样式
    index.js          项目入口文件（渲染根组件、导入组件库等）
    app.test.js       app组件测试文件
    serviceWorker.js  用来实现PWA(可选)
—————————————————————————————————————————————————————————
### 项目环境

node v20.11.0
react 18.2
react-dom  18.2.0
